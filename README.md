# PG Expense Tracker

## Description

PG Expense Tracker is a simple Java console application that helps students track their daily expenses and calculate their total spending.

## Features

* Add a new expense
* View all expenses
* Calculate total expenses
* Search expenses by category
* Delete a specific expense

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class
* Collections Framework

## Project Structure

* Expense.java - Stores expense details
* ExpenseManager.java - Handles expense operations
* Main.java - Menu-driven application

## How to Run

1. Compile the Java files:
   javac *.java

2. Run the application:
   java Main

## Sample Categories

* Food
* Travel
* Rent
* Shopping
* Entertainment
